# What's New in Version 9.0.0

**The Hybrid Era Begins**
SSL v9 introduces a revolutionary Hybrid Runtime that merges the performance of Rust with the flexibility of modern Web Technologies.

**New Features:**
*   **Native HTML & JS Integration**: Embed web interfaces directly into your system code using `html {}` and `script {}` blocks.
*   **"Ghost Window" Engine**: Build non-rectangular, perfectly shaped applications. Click-through regions allow your app to blend seamlessly with the OS desktop.
*   **16-Language Native NLP**: Write code in your native language (English, German, Spanish, French, Portuguese, and 11 more) with no translation layer.
*   **Instant HMR**: Experience lightning-fast development with Hot Module Replacement that updates your UI instantly upon saving.
*   **Asset Protocol**: Load local resources securely via `asset://` for a truly native application feel.
*   **Tailwind CSS Built-in**: Rapidly style your applications with automatic Tailwind injection.

Welcome to the future of naturally multilingual, hybrid software development.
