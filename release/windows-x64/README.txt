SSL v7.0 - Native Compilation Edition

Sonner Studio Language - The world's first programming language with:
- Native x64/ARM64 compilation (25x faster than Python)
- Natural Language Programming in 9 languages
- Non-rectangular window shapes
- Multi-architecture support

## Quick Start

```bash
# Create hello.ssl
echo 'fn main() -> Int { print("Hello!") return 0 }' > hello.ssl

# Compile and run
ssl compile hello.ssl
./hello
```

## Commands

- `ssl compile file.ssl` - Compile to native executable
- `ssl run file.ssl` - Compile and run
- `ssl check file.ssl` - Syntax check
- `ssl version` - Show version
- `ssl help` - Show help
