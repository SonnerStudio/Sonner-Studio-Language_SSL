# SSL v5.0 Binary Distribution

## Dateien

| Datei | Beschreibung |
|-------|--------------|
| `ssl.exe` | SSL Compiler und Interpreter (Windows x64) |

## Installation

1. Kopieren Sie `ssl.exe` in ein Verzeichnis Ihrer Wahl
2. Fügen Sie das Verzeichnis zu Ihrer PATH-Umgebungsvariable hinzu
3. Testen Sie mit: `ssl --version`

## Verwendung

```bash
# Programm ausführen
ssl run programm.ssl

# Interaktive REPL
ssl repl

# Syntax prüfen
ssl check programm.ssl

# Hilfe anzeigen
ssl help
```

## Systemvoraussetzungen

- Windows 10/11 (x64)
- Keine weiteren Abhängigkeiten
