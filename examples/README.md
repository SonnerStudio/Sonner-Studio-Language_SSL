# SSL v7.0 Example Programs

This directory contains example programs demonstrating SSL v7.0 features.

## Categories

### 1. Basics (`basics/`)
- `hello.ssl` - Hello World
- `variables.ssl` - Variables and types
- `functions.ssl` - Function declarations
- `control_flow.ssl` - If-else and loops

### 2. Algorithms (`algorithms/`)
- `fibonacci.ssl` - Recursive Fibonacci
- `factorial.ssl` - Factorial calculation
- `prime_sieve.ssl` - Sieve of Eratosthenes
- `bubble_sort.ssl` - Bubble sort algorithm

### 3. Data Structures (`data_structures/`)
- `linked_list.ssl` - Linked list implementation
- `stack.ssl` - Stack data structure
- `queue.ssl` - Queue implementation

### 4. Advanced (`advanced/`)
- `recursive_descent.ssl` - Parser example
- `state_machine.ssl` - Finite state machine
- `calculator.ssl` - Expression calculator

### 5. System Programming (`system/`)
- `kernel_module.ssl` - OS kernel module
- `bare_metal.ssl` - Freestanding program
- `hal_example.ssl` - Hardware abstraction

## Running Examples

```bash
# Run an example
ssl run examples/basics/hello.ssl

# Compile an example
ssl compile examples/algorithms/fibonacci.ssl
./fibonacci
```
